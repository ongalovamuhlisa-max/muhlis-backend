# Online Test Backend

Bu repository **onlayn test ilovasi uchun backend** hisoblanadi.  

## Ishlash prinsipi
- Node.js + Express ishlatilgan
- API route: `/api/tests` test ma’lumotlarini JSON formatda beradi
- CORS va dotenv qo‘llangan

## Ishga tushirish
1. `npm install` — dependencies o‘rnatish
2. `npm start` — server ishga tushadi
3. Browser yoki Postman orqali tekshirish: `http://localhost:3000/api/tests`

## Fayllar
- `server.js` — backend server kodi
- `package.json` — npm konfiguratsiyasi
- `package-lock.json` — dependencies aniqligi
- `.gitignore` — node_modules va .env faylini tashlab ketish
